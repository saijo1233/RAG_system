**RAG system для анализа документации протокола 5G.**

Целью данного проекта является создание RAG-системы, умного помощника обученного на определенном датасете,
выполняющего роль подсказки для инженера или заинтересованного человека, который хочет узнать об протоколе 5G.

**Необходимо:**
-Ollama
Модель LLM: Qwen 3-4B, Q4
База знаний: Документация 5G сделанная студентами УрФУ (my_5g_database_word), официальная документация 3GPP (my_5g_database)
Embedding модель: BGE-M3

**Как работает проект?**

Нажимаем setup.bat или вручную устанавливаем Qwen 3-4B, затем модель BGE-M3, и обучаем модель на нашем датасете.
Теперь это наш официальный помощник в среде протокола 5G.
